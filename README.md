# AI Resume Screening System

## Overview
Smart resume screening web app that compares resumes with job descriptions using NLP and ML techniques.

## Features
- Upload multiple resumes
- PDF resume parsing
- TF-IDF matching
- Cosine similarity ranking
- Skill extraction
- Missing skills detection

## Tech Stack
Python, Flask, Scikit-learn, NLP, HTML/CSS

## How to Run
pip install -r requirements.txt
python app.py
